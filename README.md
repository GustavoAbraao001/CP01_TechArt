# Checkpoint 1 – Tech Art - Game Dev

Este repositório contém o projeto da Unity desenvolvido para o Checkpoint 1 da disciplina de Tech Art. O objetivo é apresentar efeitos visuais criados com o sistema de partículas da Unity, incluindo efeitos obrigatórios e um efeito original de magia, demonstrando criatividade e domínio técnico.

## Efeitos Implementados

### 1. Efeito de Chuva
Simulação de chuva com partículas, utilizando emissão constante, variação de velocidade e `Stretched Billboards` para um visual realista das gotas. Ideal para ambientação de cenas.

### 2. Efeito de Coração (Cura)
Um efeito visual de cura, representado por corações que sobem suavemente, desvanecendo-se ao longo do tempo. Utiliza uma textura de coração customizada com transparência real, otimizada para sistemas de partículas.

### 3. Efeito de Fogueira Low Poly
Fogo estilizado com fumaça, seguindo a estética low poly. As chamas e a fumaça utilizam módulos de `Noise` para um movimento orgânico e dinâmico, criando uma fogueira visualmente atraente.

### 4. Efeito de Magia Original: Explosão Arcana
Um efeito de explosão arcana original e impactante, composto por múltiplos sistemas de partículas que trabalham em conjunto:
*   **Núcleo**: O centro de energia da explosão.
*   **Onda de Choque**: Uma expansão rápida que se propaga do núcleo, utilizando uma textura de anel.
*   **Faíscas Arcanas**: Partículas menores e rápidas que se dispersam com rastros (`Trails`).
*   **Resíduo Mágico**: Uma névoa que permanece e se dissipa lentamente, adicionando um toque místico.
